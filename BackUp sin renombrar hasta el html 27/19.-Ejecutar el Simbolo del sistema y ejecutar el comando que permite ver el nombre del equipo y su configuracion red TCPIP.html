19.- Ejecutar el Simbolo del sistema y ejecutar el comando que permite ver el nombre del equipo y su configuracion red TCP/IP

Ver el Nombre del Equipo y Configuración de Red en Windows 10

Para ver el nombre del equipo y su configuración de red TCP/IP en Windows 10, puedes utilizar el Símbolo del sistema y ejecutar el comando ipconfig /all. A continuación se muestran los pasos para hacerlo:

Pasos para Ejecutar el Símbolo del sistema

Abrir el Símbolo del sistema

  1. Presiona Win + R para abrir el cuadro de diálogo Ejecutar.
  2. Escribe cmd y presiona Enter o haz clic en Aceptar.

Ejecutar el comando para ver la configuración de red

  1. En la ventana del Símbolo del sistema, escribe ipconfig /all y presiona Enter.

Ver el nombre del equipo y la configuración de red

  1. El comando ipconfig /all mostrará información detallada sobre la configuración de red, incluyendo el nombre del equipo, direcciones IP, máscaras de subred, puertas de enlace predeterminadas y más.
